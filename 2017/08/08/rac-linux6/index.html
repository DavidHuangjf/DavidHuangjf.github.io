若表决磁盘所在磁盘组属性为EXTERNAL,为一块,若为NORMAL,则需要三块,若为HIGH,则需要五块,需要为奇数,这是为了发生脑裂时,用于表决。

1、安裝oracle RAC所需的linux軟件包
[root@note1~] # yum –y install binutils compat-libstdc++ elfutils-libelf elfutils-libelf-devel compat-libcap1
elfutils-libelf-devel-static gcc gcc-c++ glibc glibc-common glibc-devel glibc-headers ksh libaio libaio-devel
libgcclibgomp libstdc++ libstdc++-devel make sysstat unixODBC unixODBC-devel*
节点二上执行相同的命令,将所需的包安装上。

2.这里使用集群时间同步服务(CTSS),需要卸载网络时间协议(NTP)
service ntpd stop
chkconfig –level 2345 ntpd off
mv /etc/ntp.conf /etc/ntp.conf.bak
rm –rf /etc/ntp.conf
在集群安装完后,要确认ctssd是否处于活动状态
[grid@note1 ~] #crsctl check ctss

3.配置内核参数
sh configure.sh

4.创建RAC相关权限组,用户和目录
groupadd oinstall
groupadd asmadmin
groupadd asmdba
groupadd asmoper
groupadd dba
groupadd oper

useradd -g oinstall -G asmadmin,asmdba,asmoper grid
passwd grid
节点一ORACLE_SID=+ASM1
编辑grid用户的环境配置文件(~/.bash_profile),加入如下内容
export PS1=”/bin/hostname -s-> “
export TMP=/tmp
export TMPDIR=$TMP
export ORACLE_SID=+ASM1 #节点二为+ASM2,节点三为+ASM3
export ORACLE_BASE=/u/app/grid
export ORACLE_HOME=/u/app/11.2.0/grid
export ORACLE_TERM=xterm
export NLS_DATE_FORMAT=’yyyy/mm/dd hh24:mi:ss’
export TNS_ADMIN=$ORACLE_HOME/network/admin
export PATH=/usr/sbin:$PATH
export PATH=$ORACLE_HOME/bin:$PATH
export LD_LIBRARY_PATH=$ORACLE_HOME/lib:/lib:/usr/lib
exportCLASSPATH=$ORACLE_HOME/JRE:$ORACLE_HOME/jlib:$ORACLE_HOME/rdbms/jlib
umask 022

useradd -g oinstall -G dba,oper,asmdba oracle
passwd oracle
节点一:ORACLE_SID=racdb1
编辑oracle用户的环境配置文件(~/.bash_profile),加入如下内容
export TMP=/tmp
export TMPDIR=$TMP
export ORACLE_SID=racdb1 #节点二为racdb2,节点三为racdb3
export ORACLE_BASE=/u/app/oracle
export ORACLE_HOME=$ORACLE_BASE/product/11.2.0/db_1
export ORACLE_UNQNAME=racdb
export TNS_ADMIN=$ORACLE_HOME/network/admin
export ORACLE_TERM=xterm
export PATH=/usr/sbin:$PATH
export PATH=$ORACLE_HOME/bin:$PATH
exportLD_LIBRARY_PATH=$ORACLE_HOME/lib:/lib:/usr/lib:$ORACLE_HOME/jdbc/lib
exportCLASSPATH=$ORACLE_HOME/JRE:$ORACLE_HOME/jlib:$ORACLE_HOME/rdbms/jlib
export NLS_DATE_FORMAT=’yyyy/mm/dd hh24:mi:ss’
umask 022

mkdir -p /u/app/grid
mkdir -p /u/app/11.2.0/grid
chown -R grid:oinstall /u/app
mkdir -p /u/app/oracle
chown oracle:oinstall /u/app/oracle
chmod -R 775 /u
